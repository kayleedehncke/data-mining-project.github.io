# Data Mining Project

## Project Overview
This project focuses on analyzing extreme heat events across the United States to better understand their geographic and temporal variations. By examining historical temperature patterns, the project aims to identify trends in extreme heat and their implications for public health and climate resilience. The project involves data mining techniques, research question analysis, and professional communication through a public website.

## Project Milestones
1. **Milestone 1: Project Framing & Website Launch**
   - Define research questions and project goals.
   - Launch the project website.

2. **Milestone 2: Data Preparation/Collection & Cleaning**
   - Collect and clean datasets for analysis.
   - Ensure data readiness for modeling.

3. **Milestone 3: Model Implementation**
   - Implement data mining models to answer research questions.
   - Analyze results and refine models as needed.

4. **Milestone 4: Conclusion, Results & Project Report Overview**
   - Summarize findings and conclusions.
   - Prepare the final report and website updates.

## Submission Guidelines
All project materials should be submitted via Canvas in the following format:

- **Website URL**: The project website must be publicly accessible.
- **GitHub Repository URL**: Include datasets, code, and reports.
- **PDF Report**: Full project documentation in ACM format (8–12 pages).
- **ZIP File**: Containing source code and key results.
- **Team Contribution Report**: Detail individual contributions.
